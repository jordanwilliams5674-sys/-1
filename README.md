# Beidou Investment Site

Static deployment package for the Beidou investment research dashboard.

Current deployment target: Cloudflare Pages.

- Build command: leave blank
- Build output directory: `docs`
- Environment variables: leave blank

The site is research-only. It does not connect to brokers, place orders, run live trading, or provide buy/sell instructions.

See `CLOUDFLARE_PAGES_DEPLOY.md` for deployment steps.
