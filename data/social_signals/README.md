# 北斗社媒信号入口

把人工确认过的社媒原帖放到 `signals.json` 或 `signals.jsonl`，截图放到 `media/`。

示例：

```json
{
  "股票代码": "NVDA",
  "平台": "X",
  "作者": "Jensen Huang / 相关账号",
  "文字": "原帖或截图里的关键话",
  "发布时间": "2026-06-04T13:20:00+08:00",
  "原帖链接": "https://x.com/...",
  "图片": "nvda-example.png",
  "重要级别": "important",
  "备注": "社媒只做辅助信号，需等待新闻和量价确认"
}
```

说明：

- 社媒信号默认只做辅助提醒，不自动下单，不作为单独交易结论。
- `图片` 可以写 `media/` 里的文件名，页面会通过 `/social-media/文件名` 展示。
- 没有原帖链接时，也可以只记录截图和文字，但页面会标成需要人工确认。
